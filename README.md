# aurora-vision
> 一键保存网页视频，支持抖音 / 快手 / B站 / TikTok

## ✨ 功能

| 功能 | 说明 |
|------|------|
| **智能嗅探** | 自动拦截页面视频请求，无需手动复制链接 |
| **最高画质** | 动态选择可用最高画质（抖音 4K/2K/1080p 自动最优） |
| **两种方式** | 浮动按钮（悬停下载）+ 侧边栏（集中管理） |
| **进度追踪** | 下载时实时显示百分比 |
| **图集下载** | 小红书/抖音图集整批保存到文件夹 |

## 🌐 支持平台

**抖音** · **快手** · **Bilibili B站** · **TikTok**
